HOW IT WORKS:
================================================================================

1. AUTOMATIC DETECTION
   - App detects your screen size on startup
   - Calculates scale factor (e.g., 1366x768 = 0.71x scale)
   - Applies scaling to ALL controls

2. PRODUCT CARDS
   - Base size: 200x140 pixels
   - Scales up/down based on screen
   - ENTIRE card is clickable now (not just specific spots)
   - Minimum 40x40 pixels for touch screens

3. ON-SCREEN KEYBOARD
   - Base height: 300 pixels
   - Scales with screen width
   - All keys resize proportionally
   - Rebuilds when window is resized

4. TOTAL LABEL
   - Now 42pt Bold (was 32pt)
   - Subtotal/VAT: 13pt Bold (was 12pt)
   - More prominent and easier to read
