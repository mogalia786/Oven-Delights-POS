================================================================================
COMMON CONVERSIONS
================================================================================

1. FORM SIZE:
   OLD: Me.Size = New Size(800, 600)
   NEW: ResponsiveHelper.ScaleForm(Me, 800, 600)

2. BUTTON SIZE:
   OLD: .Size = New Size(200, 60)
   NEW: .Size = ResponsiveHelper.EnsureTouchTarget(ResponsiveHelper.ScaleSize(New Size(200, 60)))

3. FONT:
   OLD: .Font = New Font("Segoe UI", 14, FontStyle.Bold)
   NEW: .Font = ResponsiveHelper.CreateScaledFont("Segoe UI", 14, FontStyle.Bold)

4. POSITION:
   OLD: .Location = New Point(50, 100)
   NEW: .Location = ResponsiveHelper.ScalePoint(New Point(50, 100))

5. PANEL SIZE:
   OLD: .Size = New Size(400, 300)
   NEW: .Size = ResponsiveHelper.ScaleSize(New Size(400, 300))

6. HEIGHT/WIDTH:
   OLD: .Height = 100
   NEW: .Height = ResponsiveHelper.ScaleSize(100)

7. SPACING:
   OLD: Dim spacing = 20
   NEW: Dim spacing = ResponsiveHelper.ScaleSpacing(20)

================================================================================
EXAMPLE: COMPLETE BUTTON CONVERSION
================================================================================

BEFORE:
-------
Dim btnPay As New Button With {
    .Text = "PAY NOW",
    .Size = New Size(250, 60),
    .Location = New Point(100, 200),
    .Font = New Font("Segoe UI", 16, FontStyle.Bold),
    .BackColor = Color.Green,
    .ForeColor = Color.White
}

AFTER:
------
Dim btnPay As New Button With {
    .Text = "PAY NOW",
    .Size = ResponsiveHelper.EnsureTouchTarget(ResponsiveHelper.ScaleSize(New Size(250, 60))),
    .Location = ResponsiveHelper.ScalePoint(New Point(100, 200)),
    .Font = ResponsiveHelper.CreateScaledFont("Segoe UI", 16, FontStyle.Bold),
    .BackColor = Color.Green,
    .ForeColor = Color.White
}

================================================================================
TOUCH-FRIENDLY REQUIREMENTS
================================================================================

✅ Minimum button size: 40x40 pixels (enforced by EnsureTouchTarget)
✅ Minimum spacing: 5 pixels (enforced by ScaleSpacing)
✅ Minimum font size: 12pt for readability
✅ All sizes scale proportionally
✅ Forms don't exceed screen bounds

================================================================================
DEBUGGING TIPS
================================================================================

1. Check scale factor:
   Debug.WriteLine($"Scale: {ResponsiveHelper.ScaleFactor}")

2. Check screen size:
   Debug.WriteLine($"Screen: {ResponsiveHelper.ScreenWidth}x{ResponsiveHelper.ScreenHeight}")

3. Verify button size:
   Debug.WriteLine($"Button: {btn.Width}x{btn.Height}")

4. Look for hardcoded values:
   Search for: "New Size(", "New Point(", "New Font("
   Replace with: ResponsiveHelper methods

================================================================================
COMMON MISTAKES
================================================================================

❌ WRONG: Me.Size = New Size(800, 600)
✅ RIGHT: ResponsiveHelper.ScaleForm(Me, 800, 600)

❌ WRONG: .Height = 100
✅ RIGHT: .Height = ResponsiveHelper.ScaleSize(100)

❌ WRONG: .Font = New Font("Segoe UI", 14)
✅ RIGHT: .Font = ResponsiveHelper.CreateScaledFont("Segoe UI", 14)

❌ WRONG: For i = 0 To 3: btn.Location = New Point(i * 100, 50)
✅ RIGHT: For i = 0 To 3: btn.Location = New Point(i * ResponsiveHelper.ScaleSize(100), ResponsiveHelper.ScaleSize(50))
